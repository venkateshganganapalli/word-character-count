# 📊 Word & Character Counter

A simple web application that counts the number of words and characters in real-time as the user types.


## 🚀 Features

* Real-time word count
* Real-time character count
* Clean and simple user interface
* Instant updates on typing


## 🛠 Technologies Used

* HTML
* CSS
* JavaScript

## 📂 Project Structure

├── index.html
├── style.css
├── script.js

## ⚙️ How It Works

* User enters text in the textarea
* JavaScript listens to input events
* Character count is calculated using:

  ```js
  content.length
  ```
* Word count is calculated by splitting text based on spaces

## 💡 Example

Input: Hello Everyone! How are you?
Output:
     * Words: 5
     * Characters: 28
